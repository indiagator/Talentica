1. First line of the data file should be a general line and not relevant data. This first line will be ignored while parsing

2. Data file should be kept in a folder name product_data in the folder where the project jar is kept. No problems if the project is run straight from Netbeans

3. Run with arguments plugin can be downloaded to run the project straight from netbeans

4. Project can be opened directly from NetBeans Open Project... option

5. Processing can take longer time for files having more than 6 product offers or records
